What is a "ROM"...?

ROM emulation is the process of copying data from a ROM (read-only memory) chip to a storage medium such as a hard disk or flash memory. ... With the help of a software program called a ROM emulator, an old game can be made to run on a new computer when it won't play directly.

  1. Just download the emulator
  2. Install it
  3. Download the game and run it!
